# 데모 영상 대본

안녕하세요. 이번 프로젝트는 Streamlit으로 구현한 **롤 상식 퀴즈 앱**입니다.

첫 화면에서 학번 **2021204034**, 이름 **백다인**이 바로 보이도록 구성했습니다.

이 앱은 사용자명 기반 로그인, 캐싱, 퀴즈 기능을 포함합니다. 로그인에 성공하면 세션이 유지되고, 로그아웃 전까지 같은 사용자 상태가 이어집니다.

문제 데이터는 JSON 파일에서 불러오며, 로딩과 검증 결과를 `st.cache_data`로 캐싱했습니다. 따라서 같은 데이터를 다시 읽을 때는 전처리를 반복하지 않아 더 빠르게 동작합니다. 오른쪽 데이터 룸에서 마지막 로드 시각을 확인할 수 있습니다.

퀴즈는 **10 / 20 / 50 / 100문항** 중에서 선택할 수 있고, 타이머는 **없음 / 15초 / 30초** 중에서 정할 수 있습니다.
문제는 **챔피언 상식, 스킬/패시브, 포지션/운영, 롤 lore, 상점/아이템** 5개 카테고리에서 고르게 출제되며, 난이도는 4:4:2 비율로 섞이도록 만들었습니다.

퀴즈를 진행하면 현재 문제 번호, 카테고리, 난이도가 함께 표시됩니다. 타이머를 켠 경우 시간이 끝나면 자동으로 다음 문제로 넘어갑니다.

결과 화면에서는 최종 점수, 정답률, 시간 초과 수, 카테고리별 리포트, 난이도별 리포트, 틀린 문제 복습을 한 번에 확인할 수 있습니다. 그래서 단순 총점만이 아니라 어느 영역이 강하고 약한지도 바로 파악할 수 있습니다.
