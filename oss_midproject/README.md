# 롤 상식 퀴즈

Streamlit으로 만든 리그 오브 레전드 퀴즈 앱입니다.

## 기능
- 사용자명 기반 로그인
- `st.cache_data` 기반 문제 데이터 캐싱
- 객관식 + O/X 혼합 퀴즈
- 문항 수 선택: 10 / 20 / 50 / 100
- 타이머 선택: 없음 / 15초 / 30초
- 결과 리포트: 카테고리별 / 난이도별 성취도, 틀린 문제 복습

## 출제 범위
- 챔피언 상식
- 스킬/패시브
- 포지션/운영
- 롤 lore
- 상점/아이템

문제는 5개 카테고리에서 고르게 뽑히고, 난이도는 쉬움:보통:어려움 = 4:4:2 비율로 섞입니다.

## 실행
```bash
pip install -r requirements.txt
streamlit run app.py
```
